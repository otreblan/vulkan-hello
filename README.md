# Vulkan rasterizer

## Dependencies

* `build-essential`
* `cmake`
* `entt`
* `file`
* `glslc`
* `libassimp-dev`
* `libboost-container-dev`
* `libbullet-dev`
* `libglfw3-dev`
* `libglm-dev`
* `libstb-dev`
* `libvulkan-dev`
* `libvulkan-memory-allocator-dev`
* `pkgconf`
* `xdg-utils-cxx`

## Build
``` bash
cmake -B build
cmake --build build
```
